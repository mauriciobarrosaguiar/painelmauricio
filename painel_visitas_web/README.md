# Painel de Visitas Python

## Como rodar
```bash
pip install -r requirements.txt
streamlit run app.py
```

## Melhorias desta versão
- layout mais profissional e responsivo
- cards no Top visitas do dia, sem rolagem horizontal
- CNPJ visível nos cards
- filtros por cidade e potencial
- botão para abrir WhatsApp do cliente
- ficha rápida do cliente
- GAP mostra se comprou no mês atual
- produtos sugeridos pelos últimos 3 meses + foco da semana + mix/prioritários + maior valor faturado
- tela de importação com upload e modelos padrão
